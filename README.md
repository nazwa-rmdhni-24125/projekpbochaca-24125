# Sistem Manajemen Mahasiswa (PBO Python)

## Deskripsi
Projek ini dibuat untuk memenuhi tugas mata kuliah
**Pemrograman Berorientasi Objek (PBO)** kelas **Info 4**.
Program ini menggunakan konsep OOP seperti class, object,
method, dan modularisasi file.

## Fitur
- Menambahkan data mahasiswa
- Menampilkan seluruh data mahasiswa

## Struktur File
- `main.py` : Program utama
- `mahasiswa.py` : Class Mahasiswa
- `sistem.py` : Class SistemMahasiswa

## Cara Menjalankan
1. Pastikan Python sudah terinstall
2. Clone repository
3. Jalankan perintah:
   ```bash
   python main.py
